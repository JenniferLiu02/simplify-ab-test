This repository contains the full analysis pipeline for a within-subjects A/B/C experiment designed to evaluate two landing page messaging strategies for Simplify, an AI-powered job application platform. Participants were randomly assigned to evaluate either Page B (outcome-focused: "improve your interview chances") or Page C (efficiency-focused: "save time on applications"), each compared against a common Page A control. The study recruited 49 graduate students via Qualtrics survey and augmented the dataset with 153 GPT-generated synthetic respondents using the SyntheticData profiling pipeline. Analysis covers average treatment effects (ATE), heterogeneous treatment effects (HTE) by gender and AI usage, OLS regression with interaction terms, and statistical power calculations.
